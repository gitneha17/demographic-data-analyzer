# Demographic Data Analyzer

## 📌 Description

A Python project that analyzes demographic data extracted from the 1994 U.S. Census database using Pandas. The project answers several analytical questions related to race, education, income, working hours, and occupations.

This project was completed as part of the **freeCodeCamp Data Analysis with Python Certification**.

## 🎯 Objective

The objective of this project is to practice:

* Data loading and cleaning with Pandas
* Filtering and grouping data
* Calculating percentages and statistical insights
* Performing exploratory data analysis (EDA)

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy

## 📂 Project Structure

```text
├── adult.data.csv

└── README.md
```

## 🚀 Features

* Calculates the number of people in each race category
* Finds the average age of men
* Determines the percentage of people with a Bachelor's degree
* Compares income percentages for people with and without higher education
* Identifies the minimum work hours per week
* Finds the country with the highest percentage of people earning more than $50K
* Determines the most popular high-income occupation in India
